How I achieved the effect

This is something I was really confused about for a little bit, until it found clever solution. I am calling .header which has a higher z-index then .header-blur. Header-blur is placed directly under the header with the CSS filters that blur the contents when scrolling.

.header {
position: absolute;
z-index: 3;
top: 0;
left: 0;
width: 100.5%;
height: 44px;
line-height: 44px;
padding: 0;
color: #eee;
font-size: 20px;
text-align: center;
}

..

.header-blur {
position: absolute;
z-index: 2;
-webkit-filter: blur(3px);
-moz-filter: blur(3px);
-ms-filter: blur(3px);
-o-filter: blur(3px);
filter: blur(3px);
background: rgba(250,250,250,0.9);
top: 0;
left: 0;
width: 100%;
height: 44px;
padding: 0;
}
